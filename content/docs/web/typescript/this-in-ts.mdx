---
title: "this in TS"
description: ""
---
TypeScript types `this` to avoid context loss in callbacks and methods.
## In classes
```tsx
class Counter {
  count = 0;
  increment() {
    this.count++;
  }
}
```
## In callbacks
```tsx
class Button {
  label = "Click";
  handleClick() {
    console.log(this.label);
  }
}
const b = new Button();
setTimeout(() => b.handleClick(), 1000);
```
## Explicit `this` typing
```tsx
function greet(this: { name: string }) {
  console.log(`Hello, ${this.name}`);
}
```
