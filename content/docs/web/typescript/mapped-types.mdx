---
title: "Mapped types"
description: ""
---
Mapped types build new types by iterating keys from another type.
## Example
```tsx
type User = {
  id: number;
  name: string;
  email: string;
};
type OptionalUser = {
  [K in keyof User]?: User[K];
};
```
## Variations
```tsx
type ReadonlyUser = {
  readonly [K in keyof User]: User[K];
};
type StringifyUser = {
  [K in keyof User]: string;
};
```
## Generic template
```tsx
type PartialType<T> = {
  [K in keyof T]?: T[K];
};
```
