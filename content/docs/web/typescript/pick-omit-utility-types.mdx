---
title: "Pick and Omit"
description: ""
---
`Pick` keeps selected keys, `Omit` excludes selected keys.
## `Pick<T, K>`
```tsx
interface User {
  id: number;
  name: string;
  email: string;
  isAdmin: boolean;
}
type PublicUser = Pick<User, "id" | "name">;
```
## `Omit<T, K>`
```tsx
type UserWithoutEmail = Omit<User, "email">;
```
## Manual alternative
```tsx
type CustomUser = { id: User["id"]; name: User["name"] };
```
